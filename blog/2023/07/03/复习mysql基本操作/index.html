复习MySql基本操作

数据库复习是非常重要的，下面来回顾一下数据库常用的操作和语法。

  1. 显示数据库：
    SHOW DATABASES;

  2. 创建数据库：
    CREATE DATABASE 数据库名称;

  3. 创建数据库（如果不存在）：
    CREATE DATABASE IF NOT EXISTS 数据库名称;

  4. 删除数据库：
    DROP DATABASE 数据库名称;

  5. 删除数据库（如果存在）：
    DROP DATABASE IF EXISTS 数据库名称;

  6. 使用数据库：
    USE 数据库名称;

  7. 查询当前使用的数据库：
    SELECT DATABASE();

  8. 显示表：
    SHOW TABLES;

  9. 查看表结构：
    DESC 表名称;

  10. 创建表：
    CREATE TABLE 表名 (
    字段名1 数据类型1,
    字段名2 数据类型2,
    …
    字段名n 数据类型n
    );

  11. 数据类型示例：

    • tinyint：小整数型，占一个字节。
    • int：大整数型，占四个字节。
    • double：浮点类型，使用格式：字段名 double(总长度,小数点后保留的位数)。
    • date：日期值，只包含年月日。
    • datetime：混合日期和时间值，包含年月日时分秒。
    • char：定长字符串。
    • varchar：变长字符串。

  12. 案例：设计学生表
    需求：设计一张学生表，请注重数据类型、长度的合理性。

  13. 删除表：
    DROP TABLE 表名;

  14. 修改表名：
    ALTER TABLE 表名 RENAME TO 新的表名;

  15. 添加列：
    ALTER TABLE 表名 ADD 列名 数据类型;

  16. 修改列的数据类型：
    ALTER TABLE 表名 MODIFY 列名 新数据类型;

  17. 修改列的名称和数据类型：
    ALTER TABLE 表名 CHANGE 列名 新列名 新数据类型;

  18. 删除列：
    ALTER TABLE 表名 DROP 列名;

  20. 更新数据：
    UPDATE 表名 SET 列名1=值1, 列名2=值2, … [WHERE 条件];

  21. 删除数据：
    DELETE FROM 表名 [WHERE 条件];

  22. 查询数据：
    SELECT 字段列表 FROM 表名 WHERE 条件列表;

  23. 模糊查询：
    使用 LIKE 关键字和通配符进行模糊查询，通配符：

    • _: 代表单个任意字符。
    • %: 代表任意个数字符。

    示例：

    • 查询姓’马’的学生信息：
      SELECT * FROM stu WHERE name LIKE '马%';
    • 查询第二个字是’花’的学生信息：
      SELECT * FROM stu WHERE name LIKE '_花%';
    • 查询名字中包含 ‘德’ 的学生信息：
      SELECT * FROM stu WHERE name LIKE '%德%';

  24. 排序查询：
    使用 ORDER BY 关键字进行排序查询。

    示例：

    • 按照年龄升序排列学生信息：
      SELECT * FROM stu ORDER BY age;
    • 按照数学成绩降序排列学生信息：
      SELECT * FROM stu ORDER BY math DESC;
    • 按照数学成绩降序排列，如果数学成绩一样，再按照英语成绩升序排列：
      SELECT * FROM stu ORDER BY math DESC, english ASC;

  25. 聚合函数：

    • count(列名)：统计数量。
    • max(列名)：最大值。
    • min(列名)：最小值。
    • sum(列名)：求和。
    • avg(列名)：平均值。

    示例：

    • 统计学生总数：
      SELECT COUNT(*) FROM stu;
    • 查询数学成绩的最高分：
      SELECT MAX(math) FROM stu;

  26. 分组查询：
    使用 GROUP BY 关键字进行分组查询。

    示例：

    • 查询男同学和女同学各自的数学平均分：
      SELECT sex, AVG(math) FROM stu GROUP BY sex;
    • 查询男同学和女同学各自的数学平均分，以及各自人数：
      SELECT sex, AVG(math), COUNT(*) FROM stu GROUP BY sex;
    • 查询男同学和女同学各自的数学平均分，以及各自人数，要求分数低于70分的不参与分组：
      SELECT sex, AVG(math), COUNT(*) FROM stu WHERE math > 70 GROUP BY sex;

  27. 分页查询：
    使用 LIMIT 关键字进行分页查询。

    示例：

    • 每页显示3条数据，查询第1页数据：
      SELECT * FROM stu LIMIT 0, 3;
    • 每页显示3条数据，查询第2页数据：
      SELECT * FROM stu LIMIT 3, 3;
    • 从上面的练习推导出起始索引计算公式：
      起始索引 = (当前页码 - 1) * 每页显示的条数
